FriendFinder

Individual Project

A compatibility-based "FriendFinder" application -- basically a dating app. This full-stack site will take in results users surveys, then compare their answers with those from other users. The app will then display the name and picture of the user with the best overall match. The app uses Express, html, JQuerry and css to handle routing. The survey has 10 questions. Each answer is on a scale of 1 to 5 based on how much the user agrees or disagrees with a question. Once found the current user's most compatible friend result is displayed as a modal pop-up. The modal display both the name and picture of the closest match.
